#  School Management System

A web-based School Management System built using Flask and MySQL.  
The application allows administrators to manage student records through a simple web interface.

---

##  Project Description

This project helps schools manage student information digitally.  
Users can add, update, view, and delete student records through a web application.

The backend is built using Flask and the data is stored in a MySQL database.

---

##  Features

- Add new students
- View student records
- Update student details
- Delete student records
- Simple and user-friendly interface

---

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
